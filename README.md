#  Tic Tac Toe (Python Game)

A fun and simple **Tic Tac Toe** game built with Python.  

---

##  Features

-  Player vs Player (2-player mode)
-  Player vs Computer (basic AI)
-  Automatic win and draw detection
-  Option for replay after each match

---

##  How to Play

1. When you start the game, you'll see a menu:

2. Choose your game mode:
- **1** → Play with a friend  
- **2** → Play against the computer  
- **3** → Exit the game  

3. Take turns while playing. 

4. On your turn, mark the position you wish make sure that't an empty cell. 

5. The first player to align **three marks in a row** (horizontally, vertically, or diagonally) wins!   
If all cells are filled and no one wins, it’s a **draw** 



##  Game Rules

- Player **X** always starts first.  
- Players take turns marking available positions (1–9).  
- A player wins by completing a line of three identical marks:
- Horizontal  
- Vertical  
- Diagonal  
- If all spots are filled and there’s no winner, the match ends in a **draw**.


##  Run the Game

Make sure you have **Python** installed.
make sure to install dependencies from requirements.txt by **pip install -r requirements.txt**
